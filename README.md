# 厕所打卡器 (Toilet Tracker)

一个简单而实用的厕所打卡记录应用，帮助你追踪和分析如厕时间。

## 功能特性

- 一键开始/结束打卡记录
- 自动计算并保存每次如厕时长
- 按月份查看使用统计数据
- 生成月度时长趋势图表
- 数据本地存储，保护隐私

## 技术架构

- Java Swing 实现桌面GUI界面
- SQLite 数据库本地存储
- JFreeChart 图表生成
- Log4j2 日志系统
- JUnit 单元测试

## 安装说明

1. 确保已安装Java 11或更高版本
2. 克隆项目代码
3. 使用Maven构建项目：
   ```bash
   mvn clean package
   ```
4. 构建完成后，在target目录下找到可执行jar文件

## 使用方法

1. 双击运行jar文件启动应用
2. 点击「开始打卡」按钮开始记录
3. 完成后点击「结束打卡」按钮
4. 点击「查看统计」按钮查看历史记录和统计图表
